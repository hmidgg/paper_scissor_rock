# Real-time Rock Paper Scissors AI

A Computer Vision project using Python, OpenCV, and MediaPipe to play Rock, Paper, Scissors against an AI in real-time.

## Features
- Real-time hand gesture recognition.
- AI-driven decision engine.
- Typewriter-style visual feedback on screen.
- Video recording capability for demo sharing.

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/hmidgg/paper_scissor_rock.git
    Install the required dependencies:
    Bash

    pip install -r requirements.txt

    Run the application:
    Bash

    python3 app.py

How to play

    Show your hand to the camera (Rock, Paper, or Scissors).

    The AI will detect your move and respond.

    Press 'q' to stop the game and save the recording.
